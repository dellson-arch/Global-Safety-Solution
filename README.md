# Global Safety Solutions ERP

A professional ERP system for safety management, compliance tracking, and inventory management.

## Project Structure

- `apps/frontend`: Next.js application (Frontend)
- `apps/backend`: NestJS application (API)
- `packages/database`: Prisma database schema and client

## Tech Stack

- **Frontend**: Next.js, Tailwind CSS, Shadcn/UI, Zustand
- **Backend**: NestJS, PostgreSQL, Prisma
- **Monorepo Management**: Turborepo

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode:
   ```bash
   npm run dev
   ```
